- run npm init and create package.json
- run npm install
    - body-parser
    - express
    - mongodb
    - mongoose
    - helmet
- create server folder
    - create app.js file, our app entrypoint
    - create db.js file, our db configuration
    - create Models folder
        - create Product.js file, our mongoose Schema example
    - create Controllers folder
        - create ProductsController file
    - create Routes folder
        - create ProductsRoutes file
    - create Dockerfile for node
    - create docker-compose.yaml
- add a 'start' script to package.json
- in db.js change the mongodb url by adding the same info inserted in docker-compose
- run docker-compose build
- run docker-compose up
- to stop and remove, run docker-compose down
